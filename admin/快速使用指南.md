# 🚀 博客管理后台快速使用指南

## 📌 一分钟启动管理后台

### 方式一：一键启动（最简单）✨

```bash
双击运行：start-admin.bat
```

**自动完成：**
1. ✅ 安装依赖
2. ✅ 清理旧文件
3. ✅ 生成新文件
4. ✅ 启动服务
5. ✅ 打开浏览器

访问地址：**http://localhost:4007**

---

### 方式二：手动启动

```bash
# 1. 安装依赖
npm install

# 2. 清理并生成（可选但推荐）
npx hexo clean
npx hexo generate

# 3. 启动后台
node admin/server.js
```

## 🎯 核心功能速览

### 📝 文章管理
- **新建文章**：点击右上角"+"按钮
- **编辑文章**：点击文章列表中的"编辑"
- **发布/草稿**：编辑界面点击"发布"或"转为草稿"
- **删除文章**：编辑界面点击"删除"按钮
- **插入图片**：编辑时点击"插入图片"

### 🖼️ 图片资源
- **查看所有图片**：切换到"图片资源"标签
- **搜索图片**：在搜索框输入关键词
- **筛选图片**：按文章分类筛选
- **复制引用**：点击图片"复制"按钮获取 Markdown 代码
- **删除图片**：点击"删除"按钮

### ⚙️ 配置管理
- **站点配置**：编辑 `_config.yml`
- **主题配置**：编辑 `_config.butterfly.yml`
- **保存配置**：编辑后自动备份并保存

### 🚀 一键部署
- 点击顶部"部署网站"按钮
- 观察实时日志和进度条
- 等待部署完成提示

## ⚠️ 常见问题

### Q1: 端口被占用怎么办？
**A:** 修改 `admin/server.js` 第 15 行的 PORT 值：
```javascript
const PORT = 4008; // 改为其他端口
```

### Q2: 图片上传失败？
**A:** 检查 sharp 是否正确安装：
```bash
npm uninstall sharp
npm install sharp --build-from-source
```

### Q3: 部署失败？
**A:** 确认 Git 配置：
```bash
git remote -v          # 检查远程仓库
git config user.name   # 检查用户信息
git config user.email  # 检查邮箱
```

### Q4: 前台无法访问？
**A:** 确保管理后台已启动，并检查：
- 浏览器地址是否正确（http://localhost:4007）
- 防火墙是否拦截
- 是否有其他程序占用 4007 端口
