## 🔧 故障排查

### 问题 1：按钮不显示

**可能原因：**
- 管理后台未启动
- 浏览器缓存未更新

**解决方法：**
```bash
# 清除缓存并刷新页面
Ctrl + Shift + R (Windows)
Cmd + Shift + R (Mac)
```

---

### 问题 2：点击无反应

**可能原因：**
- JavaScript 未加载
- 浏览器阻止了弹窗

**解决方法：**
1. 检查浏览器控制台是否有错误
2. 允许该网站的弹窗权限
3. 刷新页面重试

---

### 问题 3：样式不正常

**可能原因：**
- CSS 文件未加载
- 浏览器缓存问题

**解决方法：**
```bash
# 强制刷新页面
Ctrl + F5 (Windows)
Cmd + Shift + R (Mac)
```

---

### 问题 4：个人博客打不开

**可能原因：**
- 本地博客服务未启动
- 端口不是 4000

**解决方法：**
```bash
# 启动本地博客服务
npx hexo server

# 如果端口被占用，可以指定端口
npx hexo server -p 4001
```

然后修改按钮地址为对应的端口。

## 📝 使用技巧

### 快捷操作流程

**发布文章并预览：**
```
1. 在管理后台编辑文章
2. 点击"保存"
3. 点击"🏠 个人博客"预览效果
4. 确认无误后返回管理后台
5. 点击"↑ 部署网站"
6. 点击"⚡ Actions"查看构建状态
```

**检查部署问题：**
```
1. 部署后发现异常
2. 点击"⚡ Actions"查看构建日志
3. 定位问题
4. 返回管理后台修改
5. 重新部署
```
